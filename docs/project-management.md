# Gestión del Proyecto

Para organizar el desarrollo se está utilizando la metodología Kanban a través de un tablero en Trello.

- Las tareas se dividen en características técnicas (frontend, backend, UI).
- Las tarjetas se moverán de `Todo` a `In Progress` durante el desarrollo, y a `Done` al finalizarlas.

Enlace al tablero: [https://trello.com/b/2qy6wflS/sbc-optimizer]
