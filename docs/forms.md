# El Formulario de Sugerencias

Una parte clave de los SBC es que las soluciones cambian constantemente. Por eso, añadí un formulario para que la comunidad pueda enviar sus propias recetas.

### Detalles que marcan la diferencia:
* **Control total:** He usado "componentes controlados", lo que significa que React sabe en cada milisegundo qué está escribiendo el usuario.
* **Validaciones inteligentes:** No permito que se envíen tonterías. El formulario comprueba que la media sea realista (entre 85 y 90) y avisa con mensajes visuales si falta algún dato.
* **Feedback real:** Cuando el usuario envía una receta, el botón cambia o aparece un mensaje de éxito. Quería que el usuario supiera que su acción ha funcionado.
